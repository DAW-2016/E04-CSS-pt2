Seletores

Os seletores CSS são formas de nós personalizarmos as tags html utilizadas por nós na construção da página. Abaixo há exemplos dos seletores.

  • Seletor universal

    • O seletor universal aplica os atributos escolhidos em todos os elementos da página. Perceba que o seletor é o asterisco.

    *{
        background-color: blue;
        }


  • Seletor básico

    • Escolhendo uma tag como h1, por exemplo, todos os títulos da tag h1 serão todos modificados.

    p{
        font-size: 20px;
        }


  • Seletor múltiplo

    • Caso você queira personalizar somente aqueles parágrafos que estão de uma div dentro de outra div. Usando o seletor múltiplo você consegue definir assim:

    div div p {
        font-family: Verdana;
        }


  • Aninhamento direto

    • Outro jeito de selecionar as tags que você quer personalizar é usando a lógica do aninhamento direto. Se eu digo que eu quero personalizar somente os parágrafos que são diretamente aninhados em uma div e ignorar os parágrafos que estão dentro de uma lista, que estão dentro de uma div, por exemplo, eu posso. Veja só:

    div > p {
        color: #7AC5CD;
    }


  • Classes

    • Classes são úteis quando você tem um monte de elementos onde todos devem receber o mesmo estilo. Ao invés de aplicar as mesmas regras a vários seletores, você pode simplesmente aplicar a mesma classe a todos os elementos HTML, em seguida, definir o estilo para essa classe no arquivo CSS.

    .fancy {
        font-family: Cursive;
        color: #0000CD;
    }
    <p class="fancy">Exemplo</p>
    <h3 class="fancy">Exemplo</h3>


  • IDs

    • IDs, por outro lado, são ótimos quando você tem exatamente um elemento que deve receber um certo tipo de estilo.

    #fancy {
        font-family: Cursive;
    }
    <p id="fancy">Exemplo</p>

Box model

O box model representa o modelo de caixa (literalmente) da página, isto é, como os vários container se organizam em quatro camadas. São essas o Content, Padding, Border e Margin.

        Content

    O content é o "material" real na caixa. Se estamos a falar de um elemento <p>, o "material" é o texto do parágrafo.


        Padding

    O padding é o espaçamento entre o conteúdo e a fronteira. Podemos ajustar este valor com CSS para mover a fronteira para mais perto ou mais longe do conteúdo.


        Border

    O border é a borda do elemento. É o que temos visto se tornar visível cada vez que defininimos as propriedades da borda.


        Margin

    O margin é o espaço em torno do elemento. Quanto maior a margem, mais espaço entre o nosso elemento e os elementos em torno dele. Podemos ajustar a margem para mover os elementos de HTML para mais perto ou mais longe do outro.
