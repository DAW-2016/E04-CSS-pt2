Exercícios: CSS cheat sheet

Exercício: Implemente uma página web que contenha explicação sobre as propriedades abordadas no slide do material de CSS. Para cada propriedade, sua página deverá informar:

  1. Função da propriedade. Como ela funciona?
  2. Principais valores para a propriedade
  3. Exemplos de uso da propriedade (Mínimo 2 exemplos)

Algumas propriedades...

Background:

  • Atributo background-color: define a cor de fundo de diferentes elementos. Exemplo: uma div de cor aqua

  • Atributo background-image: define a imagem de fundo de diferentes elementos. Exemplo: uma div com a imagem de pato

Fontes:

  • Atributo font-family: define a familia da fonte a ser utilizada. Exemplo: fonte Arial

    Minha fonte é Arial

  • Atributo font-style: define o estilo de uma fonte(italico,negrito,etc). Exemplo: fonte italica

    Estou italico

  • Atributo font-size: define o tamanho da fonte. Exemplo: fonte tamanho 24px

    Estou grande

Bordas:

  • Atributo border-style: define o estilo da borda de um elemento html. Exemplos: divisão com borda solida e dentro, outra divisão que com borda tracejada

    Principais valores:

    • none
    • hidden
    • dotted
    • dashed
    • solid
    • double
    • groove

  • Atributo border-width: define a grossura da borda de uma elemento.

Margem & Padding

  • Atributo margin: define as distancia entre o espaço em branco e a borda do elemento. Exemplo: com margem de 100px ao topo, 100px em baioxo, 150px a direita e 80px a esquerda

    Teste com margem

  • Atributo padding: define espaço em branco entre o conteudo do elemento e a borda do mesmo. Exemplo:

    This paragraph has a top and bottom padding of 50px, a left padding of 80px, and a right padding of 30px.

Cores

  • Atributo color: define a cor do texto de algum elemento. Exemplo:

    Eu sou azul

  • Atributo opacity: define qual opaco é a cor de um elemento. Exemplo: com opacidade de 0.3
