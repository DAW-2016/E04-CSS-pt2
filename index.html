Box Model

Todo elemento em HTML pode ser visualizado como uma caixa, sendo o papel do navegador empilhar essas caixa para criar a estrutura da página. Essas caixas podem ser customizadas em relação a espaçamento, bordas e etc. Este modelo construção recebe o nome de box model.

A quatro propriedades ligadas ao box model:

  • Margin
  • Border
  • Padding
  • Width e height

Margin

As propriedades de margem CSS definir o tamanho do espaço em branco fora da borda.

Exemplos de codigo:

.codigo{
   margin-top: 20px;
   margin-bottom: 20px;
   margin-right: 20px;
   margin-left: 20px;
/*Ou voce pode fazer: margin: 20px 20px 20px 20px;*/
}

Border

A fronteira propriedade estenográfica define todas as propriedades da borda em uma declaração. As propriedades que podem ser definidas, são (em ordem): border-width, border-style e border-color. Não importa se um dos valores acima estiver faltando, por exemplo, border: # ff0000 sólidos; é permitido.

Exemplos de codigo:

.codigo{
   border: 1px solid black;
}

Padding

The CSS padding properties define the white space between the element content and the element border. The padding clears an area around the content (inside the border) of an element.

Exemplos de codigo

.codigo{
   padding: 20px;
}

Wideth e height

A propriedade height define a altura de um elemento e widht a propriedade que define a largura de um elemento.

.codigo{
   widht : 20px;
   height: 20px;
}

Seletores

Os seletores CSS são ferramentas que fornecem meios de estilização de elementos. Há varios modos de seleção:

  • Todos os elementos
  • Grupos de elementos
  • Elementos que possuem determinado atributo
  • Etc, etc...

Selecionando por Elementos

Modo basico de seleção.

Exemplo de codigo:

h1{
   color:red
   font-family:Arial;
}

Selecionando por id

Utilizado para estilizar um elemento em específicona página.

Exemplo de codigo:

#teste{
   color:red
   font-family:Arial;
}

Selecionando por classes

Utilizado para estilizar um grupo específico de elementos na página

Exemplo de codigo:

.wad{
   color:red
   font-family:Arial;
}

Selecionando um grupo de elementos

Permite estilizar vários elementos distintos.

Exemplo de codigo:

h1, h2, p{
   color:red
   font-family:Arial;
}

Universal

Estiliza todos elementos da pagina

Exemplo de codigo:

*{
   color:red
   font-family:Arial;
}

Descendentes

Permite estilizar elementos combase na hierarquiada página.

Exemplo de codigo:

.codigo p{
   color:red
   font-family:Arial;
}

Filhos

Estiliza elementos quesão filhos de um determinado elemento.

Exemplo de codigo:

body > p{
   color:red
   font-family:Arial;
}

Adjacentes

Estiliza elementos vizinhos entre si.

Exemplo de codigo:

h3+ul{
   color:red
   font-family:Arial;
}

Pseudo Seletores Link

Seletores que são ativos em determinadas ações.

Exemplo de codigo:

a:link{
   color:red
   font-family:Arial;
}
a:visited{
   color:red
   font-family:Arial;
}
a:hover{
   color:red
   font-family:Arial;
}
a:active{
   color:red
   font-family:Arial;
}

Pseudo Seletores Paragrafos

Aplica estilos na primeira letra do paragrafo ou na primeira linha.

Exemplo de codigo:

p:first-letter{
   font-size:15px;
}
p:first-line{
   font-size:20px; }

Pseudo Seletores Before After

Adiciona conteúdo antes e depois do elemento.

Exemplo de codigo:

h1:after{
   font-size:15px;
}
h1:before{
   font-size:15px;
}

Pseudo Seletores Primeiro Filho

Estiliza o primeiro filho de um determinado elemento.

Exemplo de codigo:

body:first-child{
   font-size:15px;
}

Pseudo seletores focus

Estiliza um elemento quando ele ganha destaque na página(click,tabbing).

Exemplo de codigo:

input:focus{
   font-size:15px;
}

Atributos

Permite que os elementos a serem estilizados sejam selecionados por meio de seus atributos.

Exemplo de codigo:

a[href$=".pdf"]{
   font-size:15px;
}
