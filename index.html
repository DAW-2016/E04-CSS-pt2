Box Model

Todo elemento em HTML pode ser visualizado como uma caixa, sendo o papel do navegador empilhar essas caixa para criar a estrutura da página. Essas caixas podem ser customizadas em relação a espaçamento, bordas e etc. Este modelo construção recebe o nome de box model.

A quatro propriedades ligadas ao box model:

  • Margin
  • Border
  • Padding
  • Conteudo
