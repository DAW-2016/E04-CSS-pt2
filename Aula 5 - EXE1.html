O CSS é uma solução para separação da estrutura (HTML) e estilo (CSS) de uma página web. Alguns dos benefícios de aliar o CSS ao seu HTML são:

  • Sintaxe simplificada
  • Modularização
  • Facilidade de manutenção

Vamos mostrar algumas propriedades do CSS, dizendo o que elas fazem e dando exemplos. Caso esteja a procura de algo específico, consulte o índice ao lado.

Margem e padding

    • Essa propriedade é usada para gerar espaço em volta do conteúdo. O padding define o tamanho do espaço branco entre o elemento do conteúdo e o elemento da borda

Seletores

Toda tag HTML que usamos é também um seletor, o qual pode ser estilizado com o uso do CSS. Existem várias formas de escolher quais e de que forma você personaliza suas tags. Vamos conhecer essas estratégias.

  • Seletor universal

    • O seletor universal aplica os atributos escolhidos em todos os elementos da página.

    * {
        background-color: blue;
    }
  • Seletor básico

    • Escolhendo uma tag como p, por exemplo, você terá todos os parágrafos personalizados da forma especificada.

    p {
        font-size: 20px;
    }
  • Seletor múltiplo

    • Caso você queira personalizar somente aqueles parágrafos que estão dentro de uma div, a qual também está dentro de outra div. Você pode especificar isso usando o seletor múltiplo. Veja:

    div div p {
        font-family: Verdana;
    }
  • Aninhamento direto

    • Outro jeito de selecionar as tags que você quer personalizar é usando a lógica do aninhamento direto. Se eu digo que eu quero personalizar somente os parágrafos que são diretamente aninhados em uma div e ignorar os parágrafos que estão dentro de uma lista, que estão dentro de uma div, por exemplo, eu posso. Veja só:

    div > p {
        color: #7AC5CD;
    }
  • Classes

    • Classes são úteis quando você tem um monte de elementos onde todos devem receber o mesmo estilo. Ao invés de aplicar as mesmas regras a vários seletores, você pode simplesmente aplicar a mesma classe a todos os elementos HTML, em seguida, definir o estilo para essa classe no arquivo CSS.

    .fancy {
        font-family: Cursive;
        color: #0000CD;
    }
    <p class="fancy">Exemplo</p>
    <h3 class="fancy">Exemplo</h3>
  • IDs

    • IDs, por outro lado, são ótimos quando você tem exatamente um elemento que deve receber um certo tipo de estilo.

    #fancy {
        font-family: Cursive;
    }
    <p id="fancy">Exemplo</p>

Box-Model

O Box-Model representa as camadas em que todos os elementos da sua página (inclusive ela mesma) são divididos. São essas o Content, Padding, Border e Margin.

  • Content

    • O content é o "material" real na caixa. Se estamos a falar de um elemento <p>, o "material" é o texto do parágrafo.

  • Padding

    • O padding é o espaçamento entre o conteúdo e a fronteira. Podemos ajustar este valor com CSS para mover a fronteira para mais perto ou mais longe do conteúdo.

  • Border

    • O border é a borda do elemento. É o que temos visto se tornar visível cada vez que defininimos as propriedades da borda.

  • Margin

    • O margin é o espaço em torno do elemento. Quanto maior a margem, mais espaço entre o nosso elemento e os elementos em torno dele. Podemos ajustar a margem para mover os elementos de HTML para mais perto ou mais longe do outro.
